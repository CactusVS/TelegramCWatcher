Как использовать "TelegramCWatcher"

1. Установи Python последней версии: https://www.python.org/downloads/

2. Открой cmd(командная строка) и напиши:
   pip install telethon
   pip install aiohttp

3. Открой config.json и заполни всё:
   > api_id и api_hash - Можно получить с https://my.telegram.org/
   >> Залогинься на сайте, нажми API development tools, введи любой App title и Short name и поставь платформу Desktop, всё остальное можно оставить пустым, нажми Create application и вставь айди и хэш в конфиг

   > text_to_watch - Текст который искать
   >> ["text1", "text2"] либо ["text1"] и так далее 

   > send_to - Куда отправлять оповещение при нахождении, tg это телеграм dc это дискорд. Можно выбрать что-то одно или оба
   >> ["tg", "dc"] - оба, ["tg"] - только тг, ["dc"] - только дискорд

   > telegram_target - В какой чат в телеграмме отправлять
   >> "me" - отправлять себе в сохранённые сообщения

   > discord_webhook_url - Ссылка на дискорд вебхук для отправки сообщений в дс
   > discord_ping_text - Какой пинг использовать при отправке сообщения в дискорд

4. Запусти скрипт:
   Telegram_CWatcher.py

5. Первый раз Telegram попросит номер телефона и код авторизации в аккаунт

6. Скрипт начнёт слежку и напишет тебе, если найдёт текст из конфига text_to_watch в каком-либо посте канала, даже при обновлении сообщения

❗ Работает только с теми каналами, на которые ты подписан.
