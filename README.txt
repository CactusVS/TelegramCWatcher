Как использовать "TelegramCWatcher"

1. Установи Python последней версии: https://www.python.org/downloads/

2. Открой cmd и напиши:
   pip install telethon
   pip install aiohttp

3. Открой config.json и заполни всё:
   > api_id и api_hash - можно получить с https://my.telegram.org/
   > phone_number - свой номер телефона ( +xXxXxXxXxXxX )
   > text_to_watch - текст который искать ( ["text1", "text2"] и т.д.)
   > send_to - куда отправлять оповещение при нахождении, tg это телеграм dc это дискорд. Можно выбрать что-то одно или оба ( ["tg", "dc"] )
   > telegram_target - в какой чат в телеграмме отправлять. ( "me" , отправлять себе в сохранённые сообщения)
   > discord_webhook_url - сыллка на дискорд вебхук для отправки сообщений в дс. можно оставить пустым если сообираетесь использовать лишь телеграм
   > discord_ping_text - какой пинг использовать при отправке сообщения в дискорд, по дефолту стоит "@everyone"

4. Запусти скрипт:
   Telegram_CWatcher.py

5. Первый раз Telegram попросит код авторизации в аккаунт

6. Скрипт начнёт слежку и напишет тебе, если найдёт текст из строки text_to_watch в каком-либо посте канала.

❗ Работает только с теми каналами, на которые ты подписан.
